# YouTube 학습 노트 추출 스킬

YouTube 영상의 자막을 추출해 **정제된 원본 텍스트(.txt)** 와 **학습용 마크다운 노트(.md)** 두 파일을 생성한다. 멤버십·로그인 필요 영상을 Chrome 쿠키로 인증해 처리할 수 있다.

## 산출물 (두 파일 모두 생성)

| 파일 | 용도 |
|---|---|
| `<제목>.txt` | 타임스탬프 포함 정제 자막. 영상의 특정 부분 다시 찾을 때 검색용 |
| `<제목>-학습노트.md` | 잡담·중복 제거 후 학습 가능한 구조로 재정리한 마크다운 |

## 전제 조건

- **yt-dlp 설치**: 없으면 `brew install yt-dlp`로 설치. 의존성(deno, python, openssl)도 자동 설치됨
- **Chrome 로그인**: 멤버십 영상은 사용자가 Chrome에서 해당 멤버십 채널에 가입된 상태여야 함. yt-dlp가 `--cookies-from-browser chrome`으로 세션 쿠키를 가져옴
- **Python 3**: dedup 스크립트 실행용 (macOS 기본 제공)

Chrome이 실행 중이면 쿠키 추출이 실패할 수 있다. 그럴 땐 사용자에게 Chrome 완전 종료를 안내한다.

## 워크플로우

### 1. 환경 점검 + 메타 확인 (병렬 가능)

```bash
which yt-dlp || brew install yt-dlp
```

영상 메타 조회 — 제목·길이로 사용자에게 확인 메시지를 보여준다:

```bash
yt-dlp --cookies-from-browser chrome \
    --get-title --get-duration "<URL>"
```

### 2. 자막 추출

라이브/긴 영상이면 자동자막(`--write-auto-subs`)에 의존하는 게 일반적이다. 수동 자막이 있는 영상이면 `--write-subs`도 함께 켠다. 한국어 콘텐츠라면 `ko` 우선, 영문 콘텐츠라면 `en` 우선:

```bash
yt-dlp --cookies-from-browser chrome \
    --write-auto-subs --write-subs \
    --skip-download \
    --sub-langs ko,en \
    --convert-subs srt \
    -o "<임시작업디렉토리>/raw.%(ext)s" \
    "<URL>"
```

- `--skip-download`: 영상 본체는 안 받음 (자막만 필요)
- `--convert-subs srt`: VTT를 SRT로 변환 (정제 스크립트가 SRT 가정)
- 임시 작업 디렉토리는 `/tmp/yt-notes-<timestamp>` 같은 곳에 두고, 최종 산출물만 사용자 지정 디렉토리에 옮긴다

**자막이 아예 없는 영상이면**: 사용자에게 알리고 옵션 제시 — (1) 작업 중단, (2) `yt-dlp -x --audio-format mp3` + whisper로 전사 (느림). 임의로 진행하지 않는다.

### 3. 자막 정제 (dedup)

라이브 자동자막은 누적 표시 방식이라 prefix 중복이 심하다. 동봉된 dedup 스크립트로 정리:

```bash
python3 "<SKILL_DIR>/scripts/dedup_subs.py" \
    "<임시>/raw.ko.srt" \
    "<출력>/<제목>.txt"
```

스크립트는 prefix-dedup으로 약 5배 압축 + 타임스탬프 보존한다.

### 4. 출력 디렉토리 추론

기본 패턴: `youtube/<채널코드>/<시리즈>/`

- **채널코드**: 사용자가 이미 사용 중인 폴더가 있으면 그대로 (예: 쉬운코드 → `ezcd`). 없으면 영상 메타의 `uploader_id`를 짧게 줄여 제안하고 사용자에게 한 번 확인받는다. 한 번 정해진 매핑은 그 채널의 후속 영상에도 재사용한다
- **시리즈**: 제목에서 패턴 추출
  - "GC 1부:" → `gc`
  - "스프링 입문 3강" → `spring`
  - "1주차 OS" → `os`
  - 추출이 모호하면 사용자에게 묻는다
- **파일명**: 제목에서 OS-safe 변환 (콜론·슬래시 제거). 예: "GC 1부: GC 공부를 위해 알아야할 OS 지식" → `GC-1부-OS지식`

현 작업 디렉토리에 `youtube/` 폴더가 이미 있다면 그 안에 추가한다. 없다면 새로 만든다.

이번 호출 한정으로 사용자가 명시적 경로를 주면(예: "현 디렉토리에 youtube/ezcd/gc 만들어줘") 그대로 따른다.

### 5. 학습 노트 작성 (마크다운)

정제된 .txt를 전체 읽고 **자유형 구조**로 재구성한다. 영상 도메인(기술 강의, 인터뷰, 강연 등)에 따라 섹션 구조가 달라진다.

#### 노트 작성 원칙

1. **잡담·말꼬임 완전 제거**: 인사, 기침, 마이크 점검, 채팅창 인사, "잠시만요", "죄송합니다", 같은 말 반복 등은 모두 버린다
2. **개념을 결론부터**: 강사가 빌드업 길게 한 부분도 학습 노트에서는 결론과 근거를 먼저 배치
3. **시각 자료 텍스트 재현**: 강사가 도식·표·다이어그램을 설명했다면 마크다운 표/ASCII 도식으로 복원
4. **타임스탬프 인용은 선택적**: "자주 헷갈리는 포인트" 같이 다시 찾아볼 가치 있는 곳에만 `[01:09:00]` 같은 형식으로 남긴다
5. **시리즈물이면 위키링크**: `[[<이전편 노트명>]]` 형식으로 연결 (Obsidian 호환)
6. **인터뷰·시험 포인트 별도 표기**: 강사가 "이건 면접에서 물어볼 수 있다", "꼭 기억해야 한다"고 강조한 부분은 인용구나 굵게로 강조
7. **Q&A 자주 헷갈리는 포인트 섹션**: 영상 후반부에 시청자 Q&A가 있으면 별도 섹션으로 정리

#### 권장 (강제 아님) 마크다운 패턴

- 표: 비교·매핑·분류
- 코드 블록: 명령어, 의사 코드, 호출 흐름
- 인용구(`>`): 강사가 강조한 핵심 명제
- ASCII 도식: 메모리 구조, 호출 흐름 등 박스 다이어그램

#### 노트 구조 결정

영상 길이 1.5시간 = txt 약 90KB ≈ 학습 노트 12~16KB가 적정. 너무 길어지면 압축 부족, 너무 짧으면 핵심 누락.

도메인별 추천 섹션:
- **기술 강의**: 등장 배경 → 핵심 개념 → 동작 원리 → 자주 헷갈리는 포인트 → 다음 강의 예고
- **인터뷰**: 게스트 배경 → 주요 질문별 답변 요약 → 인사이트 정리
- **강연**: 핵심 주장 → 근거 → 사례 → 실행 방안

이건 추천일 뿐, 영상 내용에 가장 잘 맞는 구조를 새로 짜는 게 우선이다.

### 6. 결과 보고

사용자에게 다음 정보로 짧게 보고:

| 항목 | 값 |
|---|---|
| 영상 제목 | (메타에서) |
| 길이 | (예: 1:52:56) |
| 인증 | Chrome 쿠키로 통과 / 자막 무인증 가능 |
| 정제 .txt | 경로 (KB, 줄 수) |
| 학습노트 .md | 경로 (KB, 줄 수) |

이후 사용자가 검토하고 추가 요청(요약·재구성·시리즈 다음 편)을 하기를 기다린다.

## 트러블슈팅

| 증상 | 원인/해결 |
|---|---|
| `could not copy cookies` | Chrome 실행 중. 완전 종료 후 재시도 |
| `Sign in to confirm you're not a bot` | Chrome 쿠키가 만료됐거나 다른 프로필 사용 중. `--cookies-from-browser chrome:Default`로 프로필 명시 |
| 자막 다운로드 0바이트 | 영상에 자막 자체가 없음. whisper 전사 옵션 안내 |
| `--list-subs`에 ko 없음 | 영상이 한국어가 아닐 가능성. `en` 등 다른 언어 시도 |
| dedup 후에도 중복 많음 | 라이브가 아닌 영상인데 prefix-dedup이 과도하게 잘랐을 수도. 원본 SRT 함께 보존돼 있으니 그걸로 재정제 |

## 멘탈 모델

이 스킬의 핵심 가치는 **"긴 라이브 영상을 30분 학습 가능한 자료로 압축"** 하는 것이다. 단순 자막 다운로더가 아니다. 따라서:

- 자막 추출은 수단, 학습 노트가 본체
- 정제 .txt는 검색·재참조용 보조 자료
- 사용자가 노트를 보고 "이건 더 깊이 봐야겠다"고 판단했을 때 .txt에서 타임스탬프로 영상의 그 지점으로 점프할 수 있어야 함

이걸 잊지 말 것.
